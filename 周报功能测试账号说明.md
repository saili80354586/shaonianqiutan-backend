## 闭环测试流程

### 流程1：球员填写 → 教练审核（通过）

```
1. 登录球员账号（如 13800221003 张小刚）
   → 进入用户中心 → 周报 → 看到本周草稿
   
2. 点击填写周报
   → 填写训练参与、技术自评、身体心理状况等
   → 提交周报
   
3. 登录教练账号（666）
   → 进入教练后台 → 周报管理 → 看到待审核（李小强、张小刚）
   
4. 点击审核张小刚的周报
   → 查看球员自评内容
   → 填写教练评分（态度/技术/战术/知识）
   → 填写评语和建议
   → 点击「审核通过」
   
5. 登录球员账号（13800221003）
   → 收到通知「周报已审核」
   → 查看周报详情，看到教练评分和评语
```

### 流程2：球员填写 → 教练审核（退回）→ 重新提交

```
1. 球员提交周报
2. 教练审核时点击「退回修改」
3. 球员收到退回通知，查看退回原因
4. 修改后重新提交
5. 教练再次审核通过
```

### 流程3：教练发起周报

```
1. 登录俱乐部管理员或教练账号
2. 进入球队详情 → 周报管理
3. 点击「发起周报」
4. 选择周期和球员
5. 确认发起
6. 球员收到「周报已发起」通知
```

### 流程4：一键提醒

```
1. 登录教练账号
2. 进入周报管理
3. 看到有球员未提交（张小刚、刘小军草稿状态）
4. 点击「一键提醒」
5. 球员收到截止提醒通知
```

### 流程5：导出周报

```
1. 登录教练账号
2. 进入周报管理
3. 按状态筛选（全部/待审核/已通过）
4. 点击「导出CSV」
5. 下载包含所有周报数据的CSV文件
```
